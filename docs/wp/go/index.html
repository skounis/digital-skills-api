DSI Wordpress API

This document describes the Rest API of a Wordpress based implementation.

For the extention of the post types the Advanced Custom Fields and ACF to REST API plugins were used.

The base address of the API is the URL of the Wordpress website. For demonstration, the base address http://demo.com/ is assumed.

Authentication ¶

There is no need for authentication.
